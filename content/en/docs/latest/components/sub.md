## Overview

{{< release version="v0.19.0" >}}

As an example, the following shortcode displays subscript text.

<!-- markdownlint-disable MD037 -->
{{< example >}}

{{< /example >}}

## Arguments

The shortcode does not support any arguments. Provide input as inner text.
